# Lousy Movie Database (LMDB) API Endpoints

## Endpoints

| Endpoint    | Model     |
|-------------|-----------|
| /movies     | Movies    |
| /genres     | Genres    |

## `/movies`

### `GET /movies`

Get all movies.

### `GET /movies/:movieId`

Get a movie by ID.

### `POST /movies`

Create a new movie.

### `PUT /movies/:movieId`

Update a movie by ID.

### `DELETE /movies/:movieId`

Delete a movie by ID.

## `/genres`

### `GET /genres`

Get all genres.

### `GET /genres/:genreId`

Get a genre by ID.

### `POST /genres`

Create a new genre.

### `PUT /genres/:genreId`

Update a genre by ID.

### `DELETE /genres/:genreId`

Delete a genre by ID.
